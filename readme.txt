Concrete5 to WordPress automated data migration plugin

== Description ==

You have been running your website on the Concrete5 CMS platform for some time already. But once, you’ve realized that WordPress platform suits better for your needs. And here raises the question as to how to move your valuable and extensive data from Concrete5 to WordPress platform. Moreover, you want this migration to be flawless, accurate and effortless, since you may be a busy person or have coding skills deficiency.
= So, how this plugin can be useful in your migration-related challenge? =

= How does this plugin help: =
- Preview the prospective outcomes with the Free Demo Migration;
- Migration lasts from several minutes to several hours (depending on the content volume you’re moving from Concrete5 to WordPress);
- Identify the content items you want to transfer from Concrete5 to WordPress CMS (Data “Mapping” option);
- The price for complete migration depends on the content amount you want to transfer to WordPress.

= Content items you can transfer form Concrete5 to WordPress CMS platform: =
* Posts along with their date of creation, author and status;
* Attachments (images and videos);
* Tags;
* Content images;
* Users, user created date, their names and user groups;
* Comments, their creation date, status, author.

= Preserved SEO rankings after the migration =
* Migration of keywords, metadata/meta description;
* SEO aliases and friendly URLs migration.
* 301 Redirects from Concrete5 URLs to WordPress URLs

= Compatible with: Concrete5 5.4.x - 8.1.x and WordPress 3.5.x - 4.8.x. =
* Availability of 24/7 live support.
* If there is the need for custom migration, you can make a request for a CMS2CMS data migration service package.

== Installation ==

1. Download the plugin zip file.
1. Upload it to your WordPress website.
1. Go to Admin -> Plugins, find “CMS2CMS Automated [Your Source CMS platform] to WordPress Migration Plugin” click on it.
1. Register an account.
1. Provide your Source website URL address.
1. Download the Connection Bridge.
1. You'll see how a zip file named 'CMS2CMS' is downloaded. Inside of this zip file, there is 'cms2cms' folder. Extract it to your PC.
1. Using FTP client software, like Filezilla, provide your Source website access details (host, username, and password).
1. Then place the extracted ‘cms2cms’ folder to the Source website root folder (where index.php file is located).
1. Go back to the plugin and click 'Verify Connection'.
1. You’ll be redirected to CMS2CMS website in order to proceed with your migration.
1. Map the data you’re up to move from the Source website to your Target WordPress.
1. Select additional options.
1. Run Free Demo Migration to preview the results first.
1. Check how much the transfer will cost you and go directly to the Complete migration step.
